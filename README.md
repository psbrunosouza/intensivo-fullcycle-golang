# Intensivo GOLANG fullcycle

Essa pequena aplicaçao teve por objetivo o aprendizado de conceitos básicos
da linguagem GO, como:

- Structs
- Métodos
- Interfaces
- Tratamento de erros
- Variáveis
- Testes
- Test Suite (lib externa)
- Instancia de structs
